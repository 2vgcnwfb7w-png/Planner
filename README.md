# Planly GitHub Pages v3

Direkt für GitHub Pages geeignet.

## Hochladen

1. Alte Dateien im Repository löschen.
2. Den Inhalt dieser ZIP hochladen.
3. GitHub Pages kurz neu laden lassen.
4. Link öffnen.

## Enthalten

- Dashboard mit echten Daten
- Aufgaben mit Kategorie, Priorität, Datum und Status
- Schule
- Fächer
- Klausuren mit Countdown
- Noten-Tracker 0–15 Punkte
- Gesamtdurchschnitt
- Kalender-Platzhalter
- Einstellungen
- LocalStorage
- Luxuriöseres Apple-inspiriertes Design

Keine Installation nötig.
